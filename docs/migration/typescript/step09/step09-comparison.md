## Behavioral Comparison

- Existing npm entrypoints still use the same stable `scripts/*.js` paths.
- TypeScript implementations compile into `out-scripts/scripts`.
- Commands that invoke newly wrapped scripts before a normal compile now run `npm run compile:scripts` first.
- `test:check` now syntax-checks converted script outputs in `out-scripts`.
- `npm run test:unit` increased from 130 to 134 tests due to added coverage for local VSIX packaging command selection and failure handling.
- Theme generation stayed deterministic; no theme source change was introduced.
- Safe E2E stayed green with 16 passing tests before and after.
- Gated Neon E2E was not executed because it intentionally applies a real disposable workbench patch; its acknowledgement guard was covered by unit tests.
